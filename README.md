# HeroCovidCheck

Covid Check App.

# API
- https://disease.sh/v3/covid-19/all
- https://disease.sh/v3/covid-19/countries/thailand

# Install
```
git clone https://github.com/ChaiyapatOam/HeroCovidCheck.git
```
```
cd HeroCovidCheck
```
```
flutter pub get 
```
```
flutter run
```
